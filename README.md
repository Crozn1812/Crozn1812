## 1. AWS Sales Data Pipeline
<p>
  <img src="./assets/aws-pipeline.png" width="500" alt="aws-pipeline">
</p>

This project builds an event-driven AWS data pipeline.

### Main workflow
- Upload CSV files to Amazon S3
- Trigger AWS Lambda automatically
- Clean and transform data using Python and Pandas
- Store processed data in S3
- Store error records separately
- Track metadata in DynamoDB
- Monitor execution logs with CloudWatch
- Deploy infrastructure using AWS CDK

### Key Technologies
- Amazon S3
- AWS Lambda
- DynamoDB
- CloudWatch
- AWS CDK
- Python / Pandas / Pytest

### What I learned
- Event-driven architecture
- Data validation and transformation
- Serverless workflow on AWS
- Infrastructure as Code
- Logging and metadata tracking
